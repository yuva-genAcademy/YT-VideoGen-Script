Two Dominant Variants
Encoder vs. Decoder
Encoder-Only
BERT
Attention pattern — bidirectional
Every token attends to every other token — full context
Use cases
●Classification & labeling
●Semantic search & retrieval
●Document understanding
Not designed for text generation
Decoder-Only
GPT Claude
Attention pattern — causal (masked)
Each token only sees what came before — predicts next token
Use cases
●Chat & conversational AI
●Code generation & completion
●Reasoning & agentic tasks
Autoregressive generation — one token at a time
Wrong choice → Not just a performance issue — it is an architectural mismatch. Retrieval needs encoder-style. Generation needs decoder-only.
